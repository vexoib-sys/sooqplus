# موقع السوق المفتوح - موقع إعلانات مبوبة

موقع إعلانات مبوبة احترافي ومتطور مبني بـ HTML, CSS, و JavaScript.

## المميزات

### للمستخدمين العاديين:
- ✅ صفحة رئيسية مع عرض جميع الإعلانات
- ✅ نظام تسجيل الدخول وإنشاء حساب
- ✅ إضافة إعلانات جديدة (يتطلب تسجيل الدخول)
- ✅ البحث والفلترة حسب الفئات
- ✅ عرض الإعلانات في شبكة جذابة
- ✅ تصميم متجاوب يعمل على جميع الأجهزة

### للمدير:
- ✅ لوحة تحكم إدارية كاملة
- ✅ إدارة الإعلانات (عرض، تعديل، قبول، رفض، حذف)
- ✅ إدارة المستخدمين
- ✅ إدارة الفئات
- ✅ إنشاء تقارير مفصلة
- ✅ إعدادات الموقع

## كيفية الاستخدام

1. افتح ملف `index.html` في المتصفح
2. للتسجيل كمدير، استخدم البيانات أعلاه
3. للتسجيل كمستخدم عادي، اضغط على "إنشاء حساب"
4. بعد التسجيل، يمكنك إضافة إعلانات جديدة

## هيكل الملفات

- `index.html` - الصفحة الرئيسية
- `login.html` - صفحة تسجيل الدخول
- `register.html` - صفحة إنشاء حساب
- `post-ad.html` - صفحة إضافة إعلان
- `admin.html` - لوحة التحكم الإدارية
- `styles.css` - ملف التنسيقات
- `script.js` - الملف الرئيسي للوظائف
- `admin.js` - وظائف لوحة التحكم الإدارية
